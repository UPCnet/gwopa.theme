
===========
gwopa.theme
===========

In this package is implemented the theme to GWOPA PMP


Features
--------

- Css
- Javascripts
- Images
- Templates


Installation
------------

Install gwopa.theme by adding it to your buildout

.. code-block:: ini

    [buildout]

    ...

    eggs =
        gwopa.theme


and then run :command:`bin/buildout`
